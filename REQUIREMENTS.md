# 機能要件 (REQUIREMENTS.md)

`electrobun-builder` は、Electrobun アプリケーションを Windows 向けにパッケージングし、配布可能な形式（インストーラー等）に変換するためのツールです。

## 1. コア機能
- **パッケージング**: `electrobun build --os win` コマンドとの統合、または独自のビルドフローの提供。
- **インストーラー生成**:
    - **EXE (NSIS)**: 標準的なセットアップ形式の生成。
    - **MSI (WiX Toolset)**: エンタープライズ向けのインストーラー生成。
- **バイナリリソースの編集**:
    - 実行ファイルのアイコン（.ico）の書き換え。
    - バージョン情報、会社名、著作権表示の埋め込み（`rcedit` 等を使用）。

## 2. 設定
- `electrobun.config.ts` からの読み込み。
- Windows 固有の設定（プロダクトID、インストール先パス、ショートカット作成の有無など）のサポート。

## 3. CLI インターフェース
- ビルドターゲットの指定 (`--target nsis`, `--target wix`)。
- デバッグモードの提供。

## 4. 依存関係
- Bun ランタイムへの依存。
- Windows 環境での `makensis` (NSIS) および `wix.exe` (WiX) の利用。

## 5. 実装の詳細 (テクニカル)
- **設定解析**: Bun のネイティブな TypeScript インポート機能を利用（`import()`）。
- **バイナリ編集**: `rcedit` パッケージを介した Windows リソースの書き換え。
- **パッケージング**: `spawn` を用いた外部ツールの同期・非同期実行。
